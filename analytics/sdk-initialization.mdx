#### Add the SDK Dependency

Add the following dependency to your app-level `build.gradle` file

<CodeGroup>

```kotlin Kotlin
implementation("com.twinalyze:analytics:1+")
```


```java Java
implementation("com.twinalyze:analytics:1+");
```

</CodeGroup>

In your **project-level** `build.gradle` make sure the following plugins are configured:

<CodeGroup>

```kotlin Kotlin
plugins {
    id("com.android.application") version "8.2.0" apply false
    id("org.jetbrains.kotlin.android") version "1.9.10" apply false
}
```


```java Java
plugins {
    id 'com.android.application' version '8.2.0' apply false
    id 'org.jetbrains.kotlin.android' version '1.9.10' apply false
}
```

</CodeGroup>

#### Initializing Analytics SDK in Android

Initialize the **SDK** in your `Application` class to start tracking sessions, events, and insights. ( React Js )

<CodeGroup>

```kotlin Kotlin
class MyApp : Application() {
    override fun onCreate() {
        super.onCreate()
        Analytics.getInstance().initialize(
            this,
            "YOUR_API_KEY",
            "YOUR_ORG_ID”,
            true
        )
    }
}
```


```java Java
class MyApp extends Application {
    override fun onCreate() {
        super.onCreate()
        Analytics.getInstance().initialize(
            this,
            "YOUR_API_KEY",
            "YOUR_ORG_ID”,
            true
        );
    }
}
```

</CodeGroup>

> ✅ Best Practice \
> Always call `initialize()` inside your `Application` class — not inside an Activity — to ensure complete session tracking.

<CodeGroup>

```kotlin Kotlin
class MyApp : Application() {
    override fun onCreate() {
        super.onCreate()
        Analytics.initialize(
            this,
            "kh2RwGFsqNr1gh1NWWExAewbvIzYyNk",
            "twinalyze_demo_7356297484",
            true
        )
    }
}
```


```java Java
class MyApp extends Application {
    override fun onCreate() {
        super.onCreate()
        Analytics.Companion.initialize(
            this,
            "kh2RwGFsqNr1gh1NWWExAewbvIzYyNk",
            "twinalyze_demo_7356297484",
            true
        );
    }
}
```
